# Task 8 – Simple Sales Dashboard

## Objective
To create a basic interactive sales dashboard showing performance by product category, region, and month.

## Tool Used
Microsoft Power BI (Dashboard Design)

## Dataset
Superstore Sales Dataset (Sales, Profit, Region, Category, Order Date)

## Visuals Included
- Line Chart – Sales over Months
- Bar Chart – Sales by Region
- Pie Chart – Sales by Category
- Filter/Slicer for Region

## Key Insights
1. Sales increased steadily over the months.
2. West region recorded the highest sales.
3. Technology category contributed the largest revenue share.
4. Some regions showed lower overall performance.

## Outcome
This task helped understand how to design a clean dashboard and visually summarize business data for decision-making.
